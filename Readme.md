# AWS S3 Static Website Hosting 

## Project Overview

This project is a static website hosted on AWS S3, built using HTML and CSS.
It demonstrates how to create and deploy a cloud-hosted website without any backend.

**Files included:**
- `index.html` – Home page  
- `style.css` – Stylesheet  

---

## Features
- Fully static website (no backend needed)  
- Professional layout with header, hero section, and footer  
- Styled navigation and button elements  
- Can be hosted directly on **AWS S3**
